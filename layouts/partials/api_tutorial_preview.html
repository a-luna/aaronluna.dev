How To: Create a Flask API with JWT-Based Authentication
August 10, 2019
Python, Flask

Step-by-step instructions and in-depth explanations guide you through the process of creating a robust, production-quality REST API using Flask, Flask-RESTPlus, pyjwt, SQLAlchemy and other packages.

Code quality is a major focus, test coverage is created along with new functionality (using pytest). Other dev tools used: coverage, flake8, mypy and tox.
